## API host

`https://api.ossinsight.io`


## API CORS

Currently we only enable cross-origin resource sharing (CORS) for these domains:
* *.ossinsight.io
* localhost
* *.github1s.com


## API rate limit

For each IP address, the rate limit allows for up to 60 requests per minute.
