### Historical Stack Overflow Data

### Realtime Stack Overflow Data

Stack Overflow /events API Docs: https://api.stackexchange.com/docs/events

The API considers the following "events":

* posting a question
* posting an answer
* posting a comment
* editing a post
* creating a user
