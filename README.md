# Development

clone the code by:
```bash
git clone https://github.com/pingcap/ossinsight.git
```

install npm packages
```bash
cd ossinsight/;
yarn;
```

start the server:
```bash
yarn start;
```

then visit:
```
http://localhost:3000
```
